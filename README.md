# ReturnShield AI

##  Overview

ReturnShield AI is an NLP and Machine Learning project that predicts the probability of product returns based on customer reviews.

The project helps e-commerce platforms identify products with a high chance of being returned.

---

## Problem Statement

Product returns increase logistics costs, operational expenses, and customer dissatisfaction.

This project predicts the Return Risk (%) using customer review text.

---
